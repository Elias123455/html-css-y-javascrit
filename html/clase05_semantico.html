Las Etiquetas Estructurales Principales

header: Contiene la introducción o el encabezado de una página o sección. Usualmente lleva el logo, el título principal y el menú de navegación.
nav: Se usa específicamente para envolver el menú de navegación principal.
main: La etiqueta más importante. Debe contener el contenido principal y único de la página. Solo debe haber un main por página.
section: Agrupa contenido que tiene una relación temática, como "Últimas noticias", "Productos destacados", etc.
article: Para contenido independiente y auto-contenido que podría existir por sí solo, como un post de un blog, una noticia, o un comentario de un foro.
footer: Representa el pie de página de la web o de una sección. Usualmente contiene información de copyright, enlaces de contacto, etc.
aside: Para contenido secundario o tangencialmente relacionado (como una barra lateral con "artículos relacionados" o publicidad).
