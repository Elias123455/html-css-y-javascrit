Guía Interactiva de Position en CSS

La propiedad position define cómo se coloca un elemento. Nos permite sacarlo del flujo normal del documento y ubicarlo con precisión usando top, right, bottom y left.

1. position: static

Es el valor por defecto. El elemento se posiciona según el flujo normal de la página. Las propiedades top, right, bottom, left y z-index NO le afectan.

2. position: relative

El elemento se mueve relativo a su posición original. Deja un "hueco" donde debería haber estado. Lo más importante: se convierte en el "punto de anclaje" para sus hijos con `position: absolute`.

3. position: absolute

El elemento se saca del flujo normal y se posiciona relativo a su "ancestro posicionado" más cercano (un padre o abuelo con `position` diferente a `static`). Si no encuentra ninguno, se posiciona relativo al ``.

4. position: fixed

El elemento se saca del flujo normal y se posiciona relativo a la ventana del navegador. No se mueve aunque hagas scroll. ¡Ideal para menús, pop-ups o botones de "volver arriba"!

fixed
¡No me muevo!

5. position: sticky

Es un híbrido. Se comporta como `relative` hasta que haces scroll y llega a un punto definido (ej. `top: 0`), momento en el que se comporta como `fixed`. Perfecto para encabezados de sección que se "pegan" arriba.
