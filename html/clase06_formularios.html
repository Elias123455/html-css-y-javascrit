Formularios

1. La Etiqueta Principal: form

Todo formulario comienza con la etiqueta form, que actúa como un contenedor para todos los campos de entrada.

2. La Etiqueta Mágica: input

Esta es la etiqueta más versátil. El atributo type cambia completamente su comportamiento.

type="text": Un campo de texto de una sola línea.


type="email": Un campo para correos electrónicos (el navegador puede validarlo).


type="password": Un campo de contraseña que oculta los caracteres.


type="checkbox": Una casilla de verificación (para aceptar términos, por ejemplo).


type="radio": Un botón de opción. Varios radios con el mismo atributo name forman un grupo donde solo se puede seleccionar uno.

Opción 1 Opción 2

type="submit": Un botón que "envía" el formulario.

3. La mejor amiga del input: Label

La etiqueta label es crucial para la accesibilidad. Asocia un texto descriptivo a un campo del formulario.
Si haces clic en el label, el cursor se activa en el campo correspondiente.
Se conectan usando el atributo for en la label y el atributo id en el input.
