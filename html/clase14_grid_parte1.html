Definiciones de CSS Grid

Glosario Rápido de CSS Grid

display: grid;: El botón de encendido. Con esto, el chante empieza. Le dice al contenedor que se convierta en una cuadrícula y que sus hijos se van a alinear como usted mande.

grid-template-columns:: La propiedad más importante, la mera vena. Aquí usted define cuántas columnas quiere y qué tan anchas. Para esto se usa una unidad nueva y chivísima: la `fr` (unidad fraccional).

Ejemplo de grid-template-columns: 1fr 1fr 1fr; = Tres columnas del mismo tamaño. El navegador calcula solo cuánto es cada `fr` y reparte el espacio. ¡Una salvada!

Ejemplo de grid-template-columns: 2fr 1fr; = Dos columnas, la primera es el doble de ancha que la segunda. El espacio se reparte en 3 partes y la primera agarra 2.

grid-template-rows:: Es la misma vara que el de las columnas, pero para las filas. Aquí usted define qué tan altas quiere que sean las filas de su cuadrícula.

gap:: Igual que en Flexbox, es el espacio o "aire" que se le pone entre las celdas de la cuadrícula, tanto entre las filas como entre las columnas, para que los chunches no se vean todos pegados.
